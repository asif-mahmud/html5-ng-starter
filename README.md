#html5-ng-starter

Starter template for `HTML5`, `AngularJS` and `SASS`. 

##Requirements

- [NodeJS](https://nodejs.org/)
- [Gulp-Cli](http://gulpjs.com/)

##Installation

Clone this repository. And run `npm install` inside it.

##Gulp tasks
- `gulp default`
- `gulp clean`
- `gulp build`
- `gulp watch`
